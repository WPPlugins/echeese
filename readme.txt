== Description ==

The eCheese plugin allows you to create polls for your blog and gives you enhanced reports,
and unique features like cross-checking multiple polls.
You create polls from the WordPress dashboard, and then you can use a poll widget to display
it on your blog.

== Installation ==

1. Extract the plugin in the wp-content/plugins directory of your WordPress installation
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure it from the Settings->eCheese Polls panel

You can see a detailed instruction here : http://echeese.net/wordpress
